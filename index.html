The SenseHat Add On Board

The SenseHat was specifically made for the AstroPi Mission to give people the chance to code for space exploration. It comes with an 8x8 LED matrix and the sensors for humidity, accelerometer, magnetometer, temperature, barometric pressure, and a gyroscpe.


Try it out!

Use this code in the Trinket editor in the Python part of this tutorial to get your SenseHat to display a kiss for you.

                        
from sense_hat import SenseHat
]sense = SenseHat()
# Define some colours
r = (255, 0, 0)
p = (230, 0, 255)
b = (0, 255, 247)
O = (0, 0, 0)
X = (255, 255, 0)
# Get your SenseHat to give you a kiss
kiss = [
O, X, X, X, X, X, X, O,
X, O, O, O, O, O, O, X,
X, b, b, O, O, b, b, X,
X, O, O, O, O, O, O, X,
X, O, r, r, r, r, O, X,
X, O, O, p, p, O, O, X,
X, O, O, O, O, O, O, X,
O, X, X, X, X, X, X, O
]
# Display these colours on the LED matrix
sense.set_pixels(kiss)
